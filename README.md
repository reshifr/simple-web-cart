# Simple Web Cart

## Prerequisites
- PHP 8.2.18 or latest
- Composer 2.7.4 or latest
- Go 1.22.2 or latest
- SQLite 3.39.2 or latest

## Descriptions
- **data/\***: Database directory.
- **app/\***: Backend directory.
- **ui/\***: Frontend directory.

## How to run the server
```bash
$ bash -x run.sh
```
